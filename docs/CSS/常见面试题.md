## CSS 样式优先级

行内样式 > 内嵌样式 > 链接样式 > 浏览器默认样式

行内样式：写在style属性上

内嵌样式：内嵌在head标签里

链接样式：link 引入

## CSS 选择器优先级

将每条规则中，选择符对应不同分数，累计后，从左到右进行比较，大的优先级越高。

行内选择符，1000

ID选择符，100

类选择符/伪类选择符，10

元素选择符，1

!important，优先级最高

```
id选择器（#myid）> 类选择器（.myclassname）> 标签选择器（div,h1,p）> 子选择器（ul < li）> 后代选择器（li a）> 伪类选择（a:hover,li:nth-child）
```
